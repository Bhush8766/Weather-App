# 🌦 Weather App Pro

A modern, responsive **Weather Forecast Web App** built using **HTML, CSS, and JavaScript**.
This app provides real-time weather data with a beautiful UI, custom alerts, and smooth transitions.

---

## 🚀 Features

* 🌤 Real-time weather data (OpenWeather API)
* 🔍 Search any city بسهولة
* 📱 Fully responsive (Mobile, Tablet, Desktop)
* 🎨 Modern UI with gradient design
* 🌦 Dynamic weather icons
* ✅ Success messages with image
* 🔄 Smooth screen transitions
* 💧 Humidity & 🌬 Wind info

---

## 🖼 Screens

### 1️⃣ Home Screen

* App logo

### 2️⃣ Search Screen

* Stylish search bar
* Custom alert box (error/success with image)

### 3️⃣ Weather Screen

## ⚙️ Installation & Setup

1. Clone the repository:

```bash
git clone https://github.com/your-username/weather-app-pro.git
```

2. Open the project folder:

```bash
cd weather-app-pro
```

3. Run the app:

* Open `index.html` in your browser

---

## 🔑 API Setup

This app uses **OpenWeather API**

1. Go to: https://openweathermap.org/api
2. Sign up and get your API key
3. Replace this in your code:

```javascript
appid=YOUR_API_KEY
```
